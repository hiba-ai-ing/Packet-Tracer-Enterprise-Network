# Enterprise Network Infrastructure Design using Cisco Packet Tracer

## Overview

This project simulates a secure multi-site enterprise network using Cisco Packet Tracer. It was developed as part of an engineering project at ENSAJ.

The objective is to design and implement a scalable and secure network infrastructure using routing, switching, and security technologies.

---

## Features

- VLAN segmentation
- Inter-VLAN Routing
- Dynamic Routing (OSPF)
- GRE Tunnel between two sites
- NAT/PAT for Internet access
- Access Control Lists (ACL)
- Enterprise network topology

---

## Technologies Used

- Cisco Packet Tracer
- Cisco IOS
- VLAN
- OSPF
- GRE Tunnel
- NAT/PAT
- ACL
